# Papollo-_hospital-_data-_analysis













🏥 Papollo Hospitals – Work & Leads Workflow Dashboard
















📊 Project Overview
This project presents an interactive Healthcare Analytics Dashboard built using Microsoft Power BI.
The dashboard provides insights into hospital operations such as patient admissions, billing, diagnosis trends, and doctor performance.
It helps stakeholders make data-driven decisions to improve hospital efficiency and patient care.















🎯 Objectives
Analyze patient admission and discharge patterns 
Track billing and insurance amounts 
Monitor doctor feedback and performance 
Identify most common diagnoses 
Understand bed occupancy distribution













📁 Dataset Details
The dataset contains hospital records with the following columns:
Patient_ID 
Admit_Date 
Discharge_Date 
Diagnosis 
Bed_Occupancy (Private, General, ICU) 
Test (e.g., Blood Test) 
Doctor Name 
Follow-up Date 
Feedback Rating 
Billing Amount 
Health Insurance Amount 












📌 Dashboard Features






🔹 KPI Cards
Admit Date 
Discharge Date 
Follow-up Date 
Total Billing Amount 










🔹 Visualizations
Bed Occupancy Analysis (Bar Chart) 
Doctor Feedback Distribution (Donut Chart) 
Diagnosis Type Analysis (Bar Chart) 
Billing vs Insurance Trend (Line Chart) 







🔹 Filters
Patient ID slicer 
Date range filter 









🛠 Tools & Technologies Used
Microsoft Power BI 
Microsoft Excel (Data Source) 
Data Cleaning & Transformation (Power Query) 
DAX (Data Analysis Expressions) 










📈 Key Insights
Viral infections are the most common diagnosis 
Private beds have higher occupancy compared to ICU 
Billing amount shows a decreasing trend across diagnosis types 
Doctor feedback is consistently high(rating~5)
